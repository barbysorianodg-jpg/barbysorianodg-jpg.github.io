Domain & redirect

How to make barbarasoriano.com/churches redirect to this site.

Setup in Squarespace — 2 minutes

  1. Log in to Squarespace → your barbarasoriano.com site
  2. Go to Settings → Advanced → URL Redirects
  3. Click Add URL Redirect
  4. From: /churches
  5. To: https://barbysorianodg-jpg.github.io/churches/
  6. Type: 302 – Temporary
  7. Click Save

Your live site URL

  1. Direct link: barbysorianodg-jpg.github.io/churches/
  2. After redirect: barbarasoriano.com/churches → same page
  3. GitHub deploys changes automatically within ~1 minute of publishing.
